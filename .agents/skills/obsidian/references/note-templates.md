```md
---

status: draft
updated: YYYY-MM-DD
---

# <title>

Updated: YYYY-MM-DD

## Summary

## Problem

## Proposal

## Scope

## Open Questions
```


## File Naming

- Evergreen notes: `<topic>.md`
- Time-scoped notes: `YYYY-MM-DD-<topic>.md`
